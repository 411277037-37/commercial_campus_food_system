# 商業感 React 校內學餐預點餐系統

## 線上系統網址

https://411277037-37.github.io/commercial_campus_food_system/

## 技術

- React
- Vite
- GitHub Pages
- Responsive UI
- Dashboard 介面
- localStorage 資料暫存

## 主要功能

- 學生登入
- 店家登入
- 學生預點餐
- 預約取餐時間
- 查看訂單狀態
- 店家接單與完成訂單
- 菜單管理
- 餐點售完狀態
- 店家公告
- 評論與星等
- 店家回覆評論
- 銷售統計
- 學生暱稱修改
